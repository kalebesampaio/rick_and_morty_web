## Rodando projeto

Link do deploy: https://kenzie-hub-nine-topaz.vercel.app/


É necessário instalar as dependências, portanto utilize o comando abaixo para instalar tais dependências:

````
npm  install
````

Para roda o projeto:

````
npm run dev
````
